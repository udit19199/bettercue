Provider API Keys

Keys are stored in chrome.storage.local — they never leave your device and are not cloud-synced. They are only sent directly to the respective provider API when you trigger an optimization.

Token Counting

Loads the tiktoken WASM module (~1.5 MB) on first use for accurate counts. Falls back to a fast heuristic if disabled.
